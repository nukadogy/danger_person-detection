# 危险人员检测系统

## 📖 项目概述

本项目是一个基于计算机视觉的危险人员检测系统，主要用于实时识别和跟踪持有武器（棍棒、刀具）的危险人员。系统采用深度学习技术，结合状态跟踪机制，实现了稳定、可靠的危险人员检测功能。

### 🎯 核心功能
- **武器检测**：识别视频中的棍棒和刀具
- **人员检测**：检测视频中的人员
- **危险判定**：判定人员是否持有武器
- **状态跟踪**：实现防闪烁机制，避免误检
- **视频处理**：实时处理视频并标注结果

### 🛠️ 技术栈
- **检测模型**：YOLOv11n（武器检测）、YOLOv8n（人员检测）
- **图像处理**：OpenCV
- **深度学习框架**：PyTorch
- **状态管理**：自定义状态机

## 🔍 核心算法

### 1. PersonTracker 类

`PersonTracker` 是系统的核心组件，负责：
- 人员ID生成与匹配
- 武器记忆缓存（解决高速移动丢帧问题）
- 危险状态判定
- 防闪烁状态管理

### 2. 状态管理机制

- **危险判定**：连续2帧检测到危险才判定为危险
- **安全判定**：连续12帧检测到安全才判定为安全
- **武器记忆**：缓存最近6帧的武器信息

### 3. 危险判定算法

采用双重判定机制：
1. **边界框交集判定**：计算武器与人员边界框的交集面积比例
2. **中心点判定**：检查武器中心点是否在人员边界框内

## 📝 配置说明

### 状态管理参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `safe_frames_threshold` | 12 | 连续安全帧阈值 |
| `danger_frames_threshold` | 2 | 连续危险帧阈值 |
| `max_weapon_age` | 6 | 武器记忆最大年龄 |
| `weapon_conf_threshold` | 0.45 | 武器检测置信度阈值 |

## 🔄 工作流程

1. **视频读取**：读取输入视频
2. **模型推理**：使用YOLO模型检测人员和武器
3. **数据处理**：收集检测结果并进行预处理
4. **状态管理**：使用PersonTracker更新人员状态
5. **结果绘制**：在视频帧上绘制检测结果
6. **视频输出**：保存处理后的视频

## 🚧 注意事项

1. **模型性能**：
   - GPU推理速度更快，建议使用GPU
   - 低配置设备可能会出现卡顿

2. **检测精度**：
   - 受光线、角度、遮挡等因素影响
   - 小目标武器可能检测效果不佳

3. **数据集**：
   - 项目不包含完整数据集，需要自行准备
   - 数据集格式需符合YOLO格式
